# 搜索意图模式参考

## 常见查询意图类型

### 信息查询 (information)
- "什么是..."、"...是什么"、"...的定义"
- "如何..."、"...怎么做"
- "为什么..."、"...的原因"
- 特点：寻求事实、定义、解释

### 比较分析 (comparison)
- "...和...的区别"、"...vs..."
- "...哪个更好"、"...对比"
- 特点：需要对比多个对象，分析优劣

### 问题解决 (problem_solving)
- "如何解决..."、"...的方法"
- "最佳实践"、"推荐方案"
- 特点：寻求可执行的解决方案

### 趋势探索 (trend)
- "...的发展趋势"、"...的未来"
- "...最新动态"、"...的最新进展"
- 特点：关注时间维度的变化

### 深度研究 (deep_research)
- 学术论文相关查询
- 需要多来源交叉验证的复杂问题
- 特点：需要权威来源、多角度分析

## 关键词扩展策略

### 同义词扩展
- 技术术语的常见别名
- 中英文对照术语
- 缩写与全称

### 相关概念扩展
- 上位概念（更宽泛）
- 下位概念（更具体）
- 关联概念（相关领域）

### 专业术语保留
以下类型的术语应保留原文：
- 技术框架名：React, Kubernetes, TensorFlow
- 学术论文标题
- 专有名词：人名、机构名
- 行业术语：API, SDK, ROI
