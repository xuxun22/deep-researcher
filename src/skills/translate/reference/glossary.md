## 保留英文的术语

以下术语建议保留英文原文：
- 技术框架/工具名：React, Kubernetes, TensorFlow, PyTorch
- 公司/产品名：Google, OpenAI, Anthropic
- 标准/协议名：HTTP, REST, GraphQL, OAuth
- 缩写已成通用：API, SDK, CPU, GPU, AI, ML
