# 可信度评估标准

## AI 内容质量评分维度

### 1. 事实准确性 (权重 0.25)
- **高分 (0.8-1.0)**: 包含具体数据、可验证的事实、统计数字
- **中分 (0.5-0.8)**: 大部分陈述可验证，但缺乏具体数据
- **低分 (0.0-0.5)**: 多为个人观点、缺乏事实支撑

### 2. 来源引用 (权重 0.25)
- **高分**: 引用了多个可靠来源、提供参考文献链接
- **中分**: 有部分引用但不够全面
- **低分**: 无引用或引用不可靠来源

### 3. 作者权威 (权重 0.15)
- **高分**: 作者是领域专家、有学术/职业资质
- **中分**: 作者有一定相关经验
- **低分**: 匿名或无相关背景

### 4. 时效性 (权重 0.15)
- **高分**: 最近发布或更新，反映最新进展
- **中分**: 较旧但仍有一定参考价值
- **低分**: 过时信息，可能不准确

### 5. 客观性 (权重 0.10)
- **高分**: 客观中立，呈现多方观点
- **中分**: 有一定偏向但仍有事实基础
- **低分**: 明显偏见或带有强烈立场

### 6. 完整性 (权重 0.10)
- **高分**: 全面覆盖主题，考虑多方面因素
- **中分**: 覆盖主要内容但有遗漏
- **低分**: 片面或过于简略

## 综合评估规则

- 如果某个来源在「事实准确性」或「来源引用」上得分低于 0.3，即使其他维度得分高，总评分也不应超过 0.6
- 对于时效性要求高的查询（如新闻、技术更新），时效性权重应提升至 0.25
- 如果多个高分来源之间存在矛盾，应在评估中标注并降低置信度
