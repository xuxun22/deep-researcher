# 分析维度详细说明

## 1. 主题维度

### 主题聚类方法
- 基于查询关键词的语义相似度
- 同一实体/概念的不同表述归为同一主题
- 示例：
  - "GPT-4 评测" 和 "大语言模型对比" → 主题: "LLM评测"
  - "React 性能优化" 和 "前端框架对比" → 主题: "前端技术"

### 趋势判定标准
- **rising**: 最近 30 天出现次数 > 之前 30 天 × 1.5
- **stable**: 各时间段出现次数波动 < 30%
- **declining**: 最近 30 天出现次数 < 之前 30 天 × 0.5

## 2. 来源维度

### 来源分类
- 按域名类型：学术、政府、媒体、技术、社交
- 按质量评分：高(≥0.8)、中(0.5-0.8)、低(<0.5)

### 趋势指标
- 各类来源的使用比例变化
- 新发现的高质量来源
- 来源多样性（是否过度依赖某类来源）

## 3. 时间维度

### 分析粒度
- 按周：适合短期趋势（1-3个月数据）
- 按月：适合中期趋势（3-12个月数据）
- 按季度：适合长期趋势（1年以上数据）

### 季节性识别
- 某些主题可能在特定时间段集中出现
- 注意区分真正的趋势和季节性波动

## 4. 深度维度

### 研究深度指标
- 每次研究使用的来源数量
- 来源的平均质量评分
- 搜索关键词的数量和多样性
- 最终摘要的长度和详细程度
