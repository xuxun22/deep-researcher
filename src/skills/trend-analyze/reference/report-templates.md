# 趋势报告模板

## 综合趋势报告

```markdown
# 研究趋势分析报告

**分析周期**: YYYY-MM-DD 至 YYYY-MM-DD
**研究会话数**: N 次
**分析时间**: YYYY-MM-DD HH:MM

---

## 概要

[2-3 句话概括主要发现]

## 主题趋势

### 热门主题 (Rising)
- **[主题]**: 出现 N 次，趋势 ↑，首次出现于 YYYY-MM-DD
  - 相关查询: ...

### 稳定主题 (Stable)
- **[主题]**: 持续出现，平均 N 次/月

### 衰退主题 (Declining)
- **[主题]**: 出现次数下降

## 新兴话题

- **[话题]**: 首次出现于 YYYY-MM-DD，近期已出现 N 次
  - 可能的发展方向: ...

## 来源质量趋势

- 高权威性来源占比: N%
- 最常使用的来源: domain1 (N次), domain2 (N次)
- 新发现的高质量来源: ...

## 关键洞察

1. [洞察1]
2. [洞察2]
3. [洞察3]

## 建议

1. [建议后续研究方向]
2. [建议关注的领域]
3. [建议改善的方面]
```

## 简洁趋势报告（定时触发用）

```markdown
# 周度趋势摘要

**周期**: YYYY-MM-DD 至 YYYY-MM-DD
**本周研究会话**: N 次

## 本周要点
- [要点1]
- [要点2]

## 趋势变化
- [变化1]

## 建议关注
- [建议1]
```
