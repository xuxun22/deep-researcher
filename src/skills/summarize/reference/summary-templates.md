# 摘要模板

## 标准摘要结构

```markdown
## 概述

[一段话概括核心发现，150-300字]

## 关键发现

1. **[发现标题]** - [简要描述] [来源1, 来源2]
2. **[发现标题]** - [简要描述] [来源3]
3. ...

## 详细分析

### [主题1]

[详细分析内容，包含具体数据和引用]

### [主题2]

[详细分析内容]

## 不同观点

[如有争议，客观呈现各方观点]

## 信息缺口

- [未覆盖的方面1]
- [未覆盖的方面2]

## 参考来源

1. [来源标题](URL) - [简要说明]
2. ...
```

## 摘要长度指导

- **概述**: 150-300 字
- **关键发现**: 每条 50-100 字，总计 3-7 条
- **详细分析**: 500-1500 字（取决于来源数量和内容复杂度）
- **总长度**: 通常 800-2000 字

## 质量检查清单

- [ ] 所有事实陈述都有来源标注
- [ ] 不同来源的矛盾观点都已列出
- [ ] 信息缺口已明确指出
- [ ] 没有引入搜索结果之外的信息
- [ ] 语言客观中立
- [ ] 数据和统计数字准确无误
